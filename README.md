# 광진구 결빙 위험지역 예측 프로젝트

## 1. 프로젝트 개요

서울시 결빙 위험 도로 데이터를 기반으로 **광진구 내 결빙 취약 격자**를 예측하고, 위험 원인별로 유형화하여 지도에 시각화한 프로젝트입니다.

단순히 결빙 위험 여부를 분류하는 것보다, **안전구역을 기준으로 위험 클러스터의 특성이 얼마나 벗어나는지 Z-score로 표준화**하고, 그 결과를 바탕으로 결빙 취약지역을 유형별로 해석한 점에 초점을 두었습니다.

## 2. 핵심 특화점

### 1) 안전구역 기준 Z-score 표준화

일반적인 정규화가 아니라, `danger_score = 0`이고 위험 클러스터에 속하지 않는 격자를 **안전구역 기준선**으로 설정했습니다.  
이후 위험 클러스터의 기상·지표온도·시설물·유동인구 값을 안전구역 평균과 표준편차 기준으로 변환하여, 각 클러스터가 어떤 요인에서 비정상적으로 튀는지 비교했습니다.

### 2) 위험 격자의 공간 클러스터링

기존 결빙 위험 도로가 포함된 격자를 DBSCAN으로 군집화했습니다.  
이후 Queen adjacency를 활용해 위험 격자의 1칸, 2칸 인접 격자까지 위험 영향을 확산시켜 `danger_score`를 부여했습니다.

### 3) 결빙 위험 유형화

Z-score 결과에서 특징적으로 벗어나는 패턴을 기준으로 광진구 위험 격자를 다음 4개 유형으로 분류했습니다.

| 유형 | 의미 |
|---|---|
| 주야간 온도 반전형 | 지표온도는 높지만 야간 온도 조건이 불리한 지역 |
| 야간 냉각형 | 야간 지표온도와 오전 기온이 낮아 결빙 가능성이 높은 지역 |
| 구조물 냉각형 | 교량 등 구조물 영향으로 결빙 위험이 커질 수 있는 지역 |
| 교통마찰 재결빙형 | 유동인구가 많고 야간 냉각 조건이 결합된 지역 |

### 4) 지도 기반 결과 시각화

광진구 격자별 예측 위험도를 Folium 지도 위에 표시하고, 위험 유형별 레이어를 분리하여 사용자가 위험지역의 위치와 원인을 함께 확인할 수 있도록 구성했습니다.

## 3. 사용 데이터

- 서울시 100m 격자 데이터
- 서울시 결빙 위험 도로 데이터
- MODIS 기반 겨울철 지표온도 데이터
- 시간대별 기온·습도·풍속 요약 데이터
- 교량, 터널, 지하차도 등 도로 구조물 데이터
- 광진구 유동인구 및 행정동 데이터

## 4. 분석 흐름

```text
서울시 격자 생성
→ 결빙 위험 도로와 격자 매칭
→ DBSCAN 기반 위험 격자 군집화
→ 인접 격자 위험도 확산
→ 안전구역 기준 Z-score 표준화
→ 위험 클러스터 특성 해석
→ Random Forest 위험 여부 예측
→ 광진구 위험 격자 추출
→ 위험 유형별 Folium 지도 시각화
```

## 5. 모델링 결과

Random Forest 모델을 활용해 격자 단위 결빙 위험 여부를 분류했습니다.

| 지표 | 결과 |
|---|---:|
| Accuracy | 0.94 |
| 위험 클래스 Precision | 0.60 |
| 위험 클래스 Recall | 0.53 |
| 위험 클래스 F1-score | 0.56 |
| Macro F1-score | 0.77 |

Feature importance 기준으로는 `geotemp_night`, `population`, `geotemp`, `7_10_temp` 순으로 영향이 크게 나타났습니다.  
즉, 결빙 위험 예측에서 **야간 지표온도, 유동인구, 전체 지표온도, 오전 기온**이 주요 변수로 작용했습니다.

## 6. 결과 해석

본 프로젝트의 결과는 “정확한 결빙 발생 예측”이라기보다, **광진구 내 결빙 취약 후보지를 공간적으로 탐색하고 그 위험 요인을 유형별로 설명하는 분석**에 가깝습니다.

특히 안전구역 대비 Z-score 표준화를 통해 단순히 위험도가 높은 지역을 찾는 데서 끝나지 않고, 각 지역이 **온도 요인, 구조물 요인, 교통·유동 요인 중 어떤 특성 때문에 위험하게 분류되는지** 해석할 수 있도록 했습니다.
