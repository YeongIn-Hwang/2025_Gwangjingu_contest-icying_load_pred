# 광진구 결빙 위험지역 예측 프로젝트

서울시 도로 결빙 위험 데이터를 기반으로 결빙 위험 요인을 분석하고, 광진구 내 결빙 위험 가능성이 높은 격자를 예측·시각화한 프로젝트입니다. 서울 전역을 100m 단위 격자로 나눈 뒤, 기존 결빙 위험 도로와 기상·지표온도·도로 구조물·유동인구 데이터를 결합하여 Random Forest 기반 위험지역 예측 모델을 구성했습니다.

## 프로젝트 개요

본 프로젝트는 광진구 내 결빙 취약 구간을 사전에 파악하기 위한 공간 데이터 분석 프로젝트입니다. 기존 결빙 위험 도로 데이터를 단순히 표시하는 데 그치지 않고, 결빙 위험 도로가 위치한 격자의 특성을 학습하여 아직 직접 위험도로로 표시되지 않은 광진구 내 잠재 위험 격자까지 예측하는 것을 목표로 했습니다.

주요 흐름은 다음과 같습니다.

1. 서울시 행정동 경계 데이터를 기반으로 100m × 100m 격자 생성
2. 결빙 위험 도로 데이터를 격자와 매핑하여 위험 격자 라벨 생성
3. DBSCAN으로 위험 격자 군집화
4. 인접 격자까지 위험도를 확산하여 `danger_score` 생성
5. 지표온도, 시간대별 기온·습도·풍속, 교량·터널·지하차도, 유동인구 데이터 결합
6. 다중공선성 확인 후 주요 feature 선정
7. Random Forest 모델로 결빙 위험 여부 학습
8. 광진구 격자에 모델을 적용하여 위험 확률 예측
9. Folium 지도로 위험 격자, 유형별 위험지역, 학교·어린이집·도로열선 위치 시각화

## 분석 및 모델링 과정

### 1. 서울시 격자 생성

서울시 행정동 경계 데이터를 불러온 뒤, 서울 전역을 약 100m 단위 격자로 분할했습니다. 각 격자에는 `grid_id`, 자치구, 행정동 정보가 부여됩니다.

### 2. 결빙 위험 라벨 생성

결빙 위험 도로 구간을 `LineString` 형태로 변환하고, 각 격자와 교차하는 위험 도로 수를 계산했습니다. 위험 도로와 교차하는 격자는 기본 위험 격자로 간주했습니다.

이후 DBSCAN을 활용해 위험 격자의 공간적 군집을 찾고, Queen adjacency 기반으로 인접 1칸, 2칸 격자까지 위험도를 확장했습니다.

위험도는 다음과 같이 부여했습니다.

| 구분 | danger_score |
|---|---:|
| 결빙 위험 도로 포함 격자 | 2.0 |
| 1칸 인접 격자 | 1.0 |
| 2칸 인접 격자 | 0.5 |
| 그 외 격자 | 0.0 |

최종적으로 `danger_score > 0`인 격자를 결빙 위험지역으로 라벨링했습니다.

### 3. Feature 구성

초기에는 기온, 습도, 풍속, 지표온도, 구조물, 유동인구 등 다양한 변수를 사용했습니다. 이후 VIF를 통해 다중공선성을 확인하고, 최종 모델에는 다음 변수를 사용했습니다.

```text
geotemp
7_10_temp
bridge
tunnel
underpass
population
geotemp_night
```

### 4. 모델 학습

결빙 위험 여부를 예측하기 위해 Random Forest Classifier를 사용했습니다.

```python
RandomForestClassifier(
    n_estimators=200,
    random_state=42,
    class_weight='balanced'
)
```

모델은 서울시 전체 격자 데이터를 기반으로 학습되었으며, 결빙 위험/비위험 클래스 불균형을 고려하기 위해 `class_weight='balanced'`를 적용했습니다.

## 위험 유형 분류

단순 예측 확률 외에도, 광진구 위험 격자를 조건 기반으로 4가지 유형으로 나누어 해석했습니다.

| 유형 | 설명 | 조건 예시 |
|---|---|---|
| 주야간 온도 반전형 | 지표온도는 높지만 야간 지표온도가 낮아지는 지역 | `geotemp_z >= 1.0`, `geotemp_night_z <= 0.3` |
| 야간 냉각형 | 야간 지표온도와 오전 기온이 낮은 지역 | `geotemp_night_z <= -0.5`, `7_10_temp_z <= -0.5` |
| 구조물 냉각형 | 교량 등 구조물 영향으로 결빙 가능성이 높은 지역 | `geotemp_night_z <= 0`, `bridge > 0` |
| 교통마찰 재결빙형 | 유동인구가 많고 야간 냉각이 나타나는 지역 | `population_z >= 0.3`, `geotemp_night_z <= -0.3` |

각 유형은 Folium 레이어로 분리해 지도에서 확인할 수 있도록 구성했습니다.

## 한계 및 개선 방향

- 현재 위험 라벨은 실제 사고 발생 데이터가 아니라 기존 결빙 위험 도로와 인접 격자 확산 방식으로 생성되었습니다.
- 격자 중심점 계산에 위경도 좌표계가 그대로 사용되어, 정밀 분석을 위해서는 EPSG:5179 등 투영좌표계 변환 후 거리·면적 계산을 수행하는 것이 적절합니다.
- Random Forest 모델은 해석과 구현이 단순하다는 장점이 있으나, 공간 자기상관을 직접 반영하지는 않습니다.
- 향후에는 실제 제설 민원, 사고 이력, 도로 경사도, 음영 지역, 강수·적설 데이터 등을 추가하면 예측 신뢰도를 높일 수 있습니다.
- 광진구 내 도로열선, 학교, 어린이집 등 정책 우선순위 데이터를 함께 활용하면 제설 인프라 배치 의사결정에 더 직접적으로 연결할 수 있습니다.

## 요약

이 프로젝트는 서울시 결빙 위험 도로와 환경 데이터를 결합하여 결빙 위험의 공간적 패턴을 학습하고, 광진구 내 잠재 결빙 위험 격자를 예측한 분석 프로젝트입니다. 단순 지도 표시가 아니라, 위험 라벨 생성, 군집화, feature 분석, 머신러닝 예측, 정책 활용 가능한 지도 시각화까지 포함한 end-to-end 공간 데이터 분석 파이프라인을 구성했습니다.
